# its-doc — Внутреннее приложение документации ITS Agency

**its-doc** — это корпоративное приложение для хранения, чтения и редактирования внутренней документации, инструкций, примеров и демо-компонентов, разработанное для команды ITS Agency.

---

## Технологии

- **Next.js** 15 (App Router)
- **TypeScript**
- **[TailwindCSS](https://tailwindcss.com/)**
- **[Shadcn UI](https://ui.shadcn.com/docs/components)** — UI-библиотека
- **MDX** — для рендеринга инструкций из `.mdx` файлов
- **react-scan** — для анализа производительности и ошибок в dev-режиме
- **ESLint + TypeScript strict rules**

---

## Установка и запуск

### 1. Установка зависимостей

node -v 20

```
nvm use 20
```

```bash
npm install
```

или

```bash
yarn
```

### 2. Запуск разработки

```bash
npm run dev
```

или

```bash
yarn dev
```

> ⚠ Важно: проект работает на **Webpack**, не Turbopack.

---

## Особенности

- Страницы генерируются автоматически из папки `app/docs/md/` (рекурсивно)
- Файлы `index.mdx` считаются корневыми страницами разделов
- Поддержка MDX-компонентов (`mdx-components.tsx`) — можно кастомизировать теги, кнопки, и т.д.
- Подключен `react-scan` — помогает искать перформанс-проблемы и потенциальные баги во время разработки
- Используется `Shadcn UI` + `Tailwind`
